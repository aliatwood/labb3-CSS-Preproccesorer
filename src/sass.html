Sass utvärdering

Fördelar med Sass

  • Enkelt att ändra färgschema och typografi med variabler
  • CSS blir mer organiserat med nesting
  • Mixins sparar mer tid genom att duplicering av kod undviks

Nackdelar/Utmaningar med Sass

  • Kan kännas som att man måste lära sig något nytt
  • Krävs skapande av flera scss filer, medan CSS bara kräver en fil

Erfarenheter

I början kändes det något ovant med att använda scss, men efter att jag blev insatt på ämnet, blev det mycket smidigare. Att använda variabler och mixins för layout kändes ganska smidigt.
