Sass utvärdering

Fördelar med Sass

  • Enkelt att ändra färgschema och typografi med variabler
  • CSS blir mer organiserat med nesting
  • Mixins sparar mer tid genom att duplicering av kod undviks

Nackdelar/Utmaningar med Sass

  • Kan kännas som att man måste lära sig något nytt
  • Stöds inte av webbläsare, kräver att man kompilerar till vanlig CSS först.

Erfarenheter

I början kändes det något ovant med att använda scss, men efter att jag blev insatt i ämnet, blev det mycket smidigare. Att använda variabler och mixins för layout kändes ganska smidigt. Det tog lite tid att förstå allting med partials, men när det väl fungerade blev det mycket enklare att organisera koden i olika filer för variabler, mixins och komponenter.
